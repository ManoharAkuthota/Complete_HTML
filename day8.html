An HTML form is used to to collect the information/ input from the user, to create this we use form tag. This consist of different elements.

                Input Elemet: it can be type of text, character, radio, button, and submit and etc. we also have a file type.
    
                Textarea Element: Defines a multiline text, input, columns, rows attributes can be used to size the textarea. 
    
                Select elements: Defines a drop down list.
    
                type: this will defines what type of data should be entered in the box.
    
                Placeholder: This is an hint to the user, what have to enter in the box
    
                Apart of 'text' type, we should create a separate 'id' for all the input type using label tag. By this we can allocate individual properties to all the inputs.
